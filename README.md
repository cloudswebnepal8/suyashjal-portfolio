# MERN Portfolio Project

This is a personal portfolio project built using the MERN stack.  
It includes backend APIs for skills, projects, experience, and contact messages.

---

##  Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- CORS
- dotenv

---

##  Features
- Get experience data
- Get skills data
- Get project data
- Contact form message storage in MongoDB
- REST API structure

---

##  API Routes

- GET /api/experience  
- GET /api/skills  
- GET /api/projects  
- POST /api/message
